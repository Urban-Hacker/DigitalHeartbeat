Simple Digital Heartbeat is a lightweight pure HTML/CSS status page. You can easily add sections. Each tests will execute a simple bash command. If the result is 0 the test is considered as a success. If anything else is returned it is considered as a failure.

The full documentation can be found on the github page.
This message will be automatically displayed in case of one or more failure(s).

It can be customized by using basic HTML like any other text field.
